# AI Threat Modeling & Dependency Risk Review Lab

Public sanitized defensive AI lab for repo-specific threat modeling, dependency risk review, patch validation, and audit-ready remediation evidence with human approval.

## Purpose

This repository contains a controlled public baseline scaffold for a defensive, local-first AI-assisted security workflow. Codex CLI was used after the scaffold existed to create a defensive baseline review artifact. The remediation proposal was completed, the approved defensive patch was applied, post-patch validation was completed, and sanitized evidence was documented.

## Scope

This repository is a controlled public portfolio lab.

In scope:

- local defensive review;
- repo-specific threat modeling;
- dependency risk review;
- remediation planning;
- human approval;
- patch validation;
- sanitized reporting.

Out of scope:

- external scanning;
- exploitation of third-party systems;
- brute force;
- malware;
- persistence;
- evasion;
- credential theft;
- real secrets;
- private machine data;
- raw local logs.

## Workflow

1. Controlled public baseline scaffold.
2. Codex-assisted defensive baseline review.
3. Repo-specific threat model.
4. Dependency risk review.
5. Prioritized defensive findings.
6. Remediation plan.
7. Human approval gate.
8. Patch validation.
9. Sanitized audit-ready reporting.
